# Job Application Web App (Spring Boot)

A simple and efficient web application built using **Java Spring Boot**, where **admins can post job vacancies** and **users can view available job openings**.  
This project demonstrates CRUD operations, MVC architecture, form handling, and database integration.

---

## Features

### Admin
- Add new job vacancies  
- Edit or update existing job posts  
- Delete job openings  
- Manage job listings in the admin panel  

### User
- View all job vacancies  
- Search and browse jobs  
- View job details  

---

## Tech Stack

- **Java 17 / 11**
- **Spring Boot**
- **Spring MVC**
- **Spring Data JPA**
- **Thymeleaf** (or your UI framework)
- **MySQL Database**
- **Hibernate**
- **Maven**
